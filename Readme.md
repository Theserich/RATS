# RAdiocarbon Table Software

A desktop application built using PyQt5 for data visualization and analysis, equipped with interactive plotting capabilities.

## **Features**

- **Direct database access for MICADAS AMS data**
- **Interactive Plotting**: Support for generating dynamic and responsive plots using Matplotlib.
- **Data Management**: Export data seamlessly.
- **Excel File Support**: Easily export data to Excel.
- **Customizable Visualizations**: Adjust plot curves, labels, and style to suit specific requirements.

## **Installation Guide**

### **Requirements**
- Python 3.13.0+ (Ensure your Python version is up-to-date to avoid compatibility issues.)
- Dependencies:
  - PyQt5
  - PyQtGraph
  - Matplotlib
  - Other dependencies defined in the `requirements.txt` file.

### **Steps to Install**
1. Clone the repository:
   ```bash
   git clone https://github.com/Theserich/Wigglematcher.git
   cd Wigglematcher
   ```

2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
 
   If additional packages are required beyond those mentioned in `requirements.txt`, ensure to install them as prompted.

## **Usage Instructions**

1. After setting up the environment, run the application using:
   ```bash
   python main.py
   ```

2. Explore the following key functionalities:
   - Show data in Table
   - Plot data interactively.
   - Customize visual properties of charts and graphs.
